Copy Router Config


作者: muts
License: GPLv2

简介

Copy-router-config是一款通过SNMP协议来获取Cisco设备的配置信息的工具（这是我见过的最简单的简介。。。）

项目主页
kali上的项目分支

copy-router-config中包含的工具

  • copy-router-config.pl
  • merge-router-config.pl

使用示例

  1. 从路由器上（IP:192.168.1.1)下载配置文件到TFTP服务器（IP:192.168.1.15）上,并通过社区认证的方式private
  1. 从路由器（IP:192.168.1.1）上合并配置文件，并复制到TFTP服务器（IP:192.168.1.15）上，是用社区认证
